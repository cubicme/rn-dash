<task type="auto">
  <name>Task 1: Fix pane title color and metro-active row highlight</name>
  <files>src/ui/panels.rs</files>
  <action>
Fix two visual issues in panels.rs:

**Pane title color (UAT Test 2):**
In all four pane rendering functions (render_worktree_table, render_metro_pane, render_log_panel, render_command_output), the title inherits border_style which is DarkGray for inactive panes, making titles unreadable. Fix by using `.title_style(Style::default().fg(Color::White))` on the Block builder. This ensures titles are always white regardless of border focus state. ratatui 0.30 supports `.title_style()` on Block.

**Metro-active row highlight (UAT Test 4):**
The current row_style sets `bg(Color::DarkGray)` for metro-active rows, but this is the same as the cursor highlight (row_highlight_style). The DarkGray bg also may not render on terminals with dark backgrounds. Fix:
1. Change metro-active row style to use `fg(Color::Green).add_modifier(Modifier::BOLD)` (green text + bold) — visible on any terminal without relying on background color
2. Keep the `●` metro icon in the status column as Green too
3. Change row_highlight_style (cursor) to use `bg(Color::DarkGray).add_modifier(Modifier::BOLD)` (unchanged, it works for cursor since Table applies it only to the selected row)

This makes metro-active rows stand out via green text color regardless of cursor position.
