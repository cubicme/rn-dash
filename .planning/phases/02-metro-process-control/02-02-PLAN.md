**IMPORTANT DESIGN REVISION:** MetroHandle needs to own the `tokio::process::Child` for kill operations. But Child is consumed by stdin/stdout handling. The solution from the research: take stdout/stderr/stdin immediately after spawn, keep Child (minus those handles) for kill. The Child still has the process handle for kill even after stdout/stderr/stdin are taken.

So MetroHandle becomes (this should have been done in Plan 01, but if not, update here):
Add a field `child: tokio::process::Child` to MetroHandle (or hold it in a separate field accessible during stop).

Actually, the simplest approach: do NOT store Child in MetroHandle. Instead, the stream_task should hold the Child. When we need to kill, we abort the stream_task (which drops Child, triggering kill_on_drop) and then verify port is free. But kill_on_drop sends SIGKILL without waiting — we need the port-free polling.

FINAL APPROACH: Store the Child PID in MetroHandle. Use a kill channel:
- MetroHandle has a `kill_tx: tokio::sync::oneshot::Sender<()>`
- The spawn task holds the Child and listens on `kill_rx`
- When kill_tx is sent, the spawn task kills the child and sends MetroExited

This is cleaner. The background task owns the Child and responds to kill signals.

**Implementation: spawn_metro_task() helper function (add to app.rs):**

```rust
async fn spawn_metro_task(
    worktree_path: PathBuf,
    filter: bool,
    action_tx: tokio::sync::mpsc::UnboundedSender<Action>,
    handle_tx: tokio::sync::mpsc::UnboundedSender<MetroHandle>,
) {
    let client = TokioProcessClient;
    match client.spawn_metro(worktree_path.clone(), filter).await {
        Ok(mut child) => {
            let pid = child.id().unwrap_or(0);
            let stdout = child.stdout.take().expect("stdout piped");
            let stderr = child.stderr.take().expect("stderr piped");
            let stdin = child.stdin.take().expect("stdin piped");

            let (stdin_tx, stdin_rx) = tokio::sync::mpsc::unbounded_channel();
            let stdin_task = tokio::spawn(stdin_writer(stdin, stdin_rx));

            let (kill_tx, kill_rx) = tokio::sync::oneshot::channel::<()>();

            let stream_tx = action_tx.clone();
            let stream_task = tokio::spawn(metro_process_task(child, stdout, stderr, kill_rx, stream_tx));

            let worktree_id = worktree_path.file_name()
                .map(|n| n.to_string_lossy().to_string())
                .unwrap_or_else(|| "unknown".to_string());

            let handle = MetroHandle {
                pid,
                worktree_id,
                stdin_tx,
                stream_task,
                stdin_task,
                kill_tx: Some(kill_tx), // wrapped in Option for take()
            };
            let _ = handle_tx.send(handle);
        }
        Err(e) => {
            tracing::error!("metro spawn failed: {e}");
            let _ = action_tx.send(Action::MetroExited);
        }
    }
}
```

**metro_process_task() — owns Child, handles kill and log streaming:**

```rust
async fn metro_process_task(
    mut child: tokio::process::Child,
    stdout: tokio::process::ChildStdout,
    stderr: tokio::process::ChildStderr,
    kill_rx: tokio::sync::oneshot::Receiver<()>,
    tx: tokio::sync::mpsc::UnboundedSender<Action>,
) {
    let log_tx = tx.clone();
    let log_task = tokio::spawn(stream_metro_logs(stdout, stderr, log_tx));

    tokio::select! {
        _ = kill_rx => {
            // Kill signal received — kill child and wait for port
            log_task.abort();
            if let Err(e) = child.kill().await {
                tracing::error!("metro kill failed: {e}");
            }
            // Poll port free (Pattern 3 from research)
            for _ in 0..50 {
                if crate::infra::port::port_is_free(8081) { break; }
                tokio::time::sleep(std::time::Duration::from_millis(100)).await;
            }
            let _ = tx.send(Action::MetroExited);
        }
        _ = child.wait() => {
            // Child exited on its own (external kill, crash)
            log_task.abort();
            let _ = tx.send(Action::MetroExited);
        }
    }
}
```

**stream_metro_logs() — reads stdout+stderr lines (Pattern 4 from research):**

```rust
async fn stream_metro_logs(
    stdout: tokio::process::ChildStdout,
    stderr: tokio::process::ChildStderr,
    tx: tokio::sync::mpsc::UnboundedSender<Action>,
) {
    use tokio::io::{AsyncBufReadExt, BufReader};
    let mut stdout_lines = BufReader::new(stdout).lines();
    let mut stderr_lines = BufReader::new(stderr).lines();

    loop {
        tokio::select! {
            line = stdout_lines.next_line() => {
                match line {
                    Ok(Some(l)) => { let _ = tx.send(Action::MetroLogLine(l)); }
                    _ => break,
                }
            }
            line = stderr_lines.next_line() => {
                match line {
                    Ok(Some(l)) => { let _ = tx.send(Action::MetroLogLine(l)); }
                    _ => break,
                }
            }
        }
    }
}
```

**stdin_writer() — forwards bytes to metro stdin (Pattern 5 from research):**

```rust
async fn stdin_writer(
    mut stdin: tokio::process::ChildStdin,
    mut rx: tokio::sync::mpsc::UnboundedReceiver<Vec<u8>>,
) {
    use tokio::io::AsyncWriteExt;
    while let Some(bytes) = rx.recv().await {
        if let Err(e) = stdin.write_all(&bytes).await {
            tracing::warn!("metro stdin write failed: {e}");
            break;
        }
    }
}
```
